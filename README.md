# Prospect Source 


## Getting up and running: 

### Hipster one-liner:
```
curl -0 https://raw.githubusercontent.com/joshmccall221/todo-react-redux/master/goScript | bash
```

### Copy Pasta:
```
git clone https://github.com/joshmccall221/todo-react-redux.git && \
cd todo-react-redux && \
nvm i 6 && \
nvm use 6 && \
npm i
```
